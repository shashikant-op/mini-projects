## âš¡ Installation

### Backend

```bash
# Navigate to backend folder
cd backend

# Install dependencies
npm install

# Start backend server
node index.js
````

Backend runs at: `http://localhost:8080`

### Frontend

```bash
# Navigate to frontend folder
cd frontend

# Install dependencies
npm install

# Start frontend in development mode
npm run dev
```

Frontend runs at: `http://localhost:3000`
